# 정책

## 1. 진행 방향

- 전체 자동번역 방식 제외 (비용 및 품질 이슈)
- **AI 번역 기반 포르투갈 특화 페이지 방식**으로 진행 (2안)
- 포르투갈 콘텐츠는 인트로 페이지 중심 노출
- 글로벌 서비스와 분리된 전용 진입 구조

---

## 2. 언어 적용 방식

- 언어 선택 기능 대신 `?language=pt` **파라미터 기반**으로 동작
- 파라미터가 붙은 상태에서만 포르투갈어 노출
- 글로벌 기능(검색 등)은 영어 유지

### 파라미터 유지 페이지

| 페이지 | 파라미터 유지 |
|--------|:---:|
| 인트로 페이지 | ✅ |
| 회차리스트 | ✅ |
| 뷰어 | ✅ |
| 로그인/회원가입 | ✅ |
| 결제 모달 | ✅ |
| 플랜 페이지 | ✅ |
| 내서재 | ✅ |
| 그 외 (검색 등) | ❌ 영문 노출 |

---

## 3. 번역 적용 범위

### Block 1
- 인트로 페이지
- 작품 콘텐츠 번역 (작품명, 줄거리, 회차 제목, 소설 본문)

### Block 2
- 회차리스트
- 뷰어
- 로그인/회원가입

### Block 3
- 결제 모달
- 플랜 페이지
- 내서재

### 번역 제외 항목
- 작가명
- 날짜/시간 데이터

---

## 4. 번역 플로우

```
포르투갈 인트로 진입
        ↓
?language=pt 파라미터 붙음
        ↓
파라미터 유지 페이지 → 포르투갈어 노출
그 외 이동           → 영문 노출
```

---

## 5. 회원 및 계정 정책

- 계정은 글로벌 노벨피아와 **공통 사용** (포르투갈 ↔ 글로벌 동일 계정)
- 가입 수단: 이메일, SNS 로그인 (OAuth)
- 닉네임: 영문 그대로 부여

### 가입 출처 기록
- `data_member.local_code` 활용
- `?language=pt` 상태에서 가입 시 → `local_code = pt` 저장

### 유입 경로 기록
- `temp_access.url` 컬럼 활용
- `?language=pt` 파라미터 포함 여부로 포르투갈 유입 확인

---

## 6. 내서재 언어 드롭다운 정책

- **기본값**: 회원의 `iso_country` 또는 `local_code` 기준
  - pt 회원 → Portuguese 우선 노출
  - 그 외 회원 → English 기본 노출
- **Portuguese 작품이 0개일 경우** → English로 자동 전환 (탭 구분 없이 동일 적용)
- **사용자가 직접 변경한 경우** → 선택값 저장, 재진입 시 유지
  - 단, Portuguese 선택 상태에서 Portuguese 작품이 0개이면 자동 English 전환 우선 적용

### 언어 옵션 표기
| 환경 | English | Portuguese |
|------|---------|------------|
| PC | English | Portuguese |
| 모바일 | EN | PT |
