### 01 | 언어 선택 드롭다운

#### 동작 방식
- 언어 변경 시 선호작 / 최근 본 작품 리스트가 해당 언어 기준으로 전환

| 선택 언어 | 노출 기준 |
|-----------|---------|
| Portuguese | Portuguese 번역 데이터가 있는 작품만 노출 |
| English | English 작품 기준으로 노출 |

#### 기본값 정책

| 회원 유형 | 기본값 | 예외 |
|-----------|--------|------|
| pt 회원 (`local_code = pt`) | Portuguese | Portuguese 작품 0개일 경우 → English 자동 전환 |
| 그 외 회원 | English | - |

#### 사용자 직접 변경 시
- 선택값 저장 → 재진입 시에도 마지막 선택 언어 유지
- 단, Portuguese 선택 상태에서 Portuguese 작품이 0개이면 **자동 English 전환 우선 적용**

#### 언어 표기

| 환경 | English | Portuguese |
|------|---------|------------|
| PC | English | Portuguese |
| 모바일 | EN | PT |

---

### 02 | 내서재 띠배너

- 내서재 상단에 포르투갈 인트로 페이지 유입용 띠배너 추가
- 어드민에서 설정
- 배너 클릭 시 → 포르투갈 인트로 페이지(`?language=pt`)로 이동
- 디자인: 인트로 페이지 배너와 통일
